======================================== Git init push, V1, V2 ======================================

1. Initialize Git inside your project
cd your_project/
git init
git add .
git commit -m "MVP Version 1 stable"

2. Create a GitHub repo and push
git remote add origin <repo-url>
git branch -M main
git push -u origin main


Now your v1 is safely stored.
